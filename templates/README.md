# 数据文件模板说明

本目录提供了各个数据类别的 Markdown 文件模板示例，用于指导如何创建新的数据文件。

## 📁 模板列表

- `news-template.md` - 行业资讯模板
- `tool-template.md` - AI 工具模板
- `prompt-template.md` - 提示词模板
- `project-template.md` - 项目案例模板
- `resource-template.md` - 学习资源模板
- `learning-journal-template.md` - 学习记录模板

## 📝 通用格式

所有数据文件都遵循以下格式：

1. **Front Matter** - 位于文件开头的元数据（使用 YAML 格式）
2. **Markdown 内容** - 人类可读的正文内容
3. **嵌入 JSON** - 位于 HTML 注释中的结构化数据（便于程序解析）

## 🔧 使用方式

1. 复制对应的模板文件
2. 修改 Front Matter 中的字段
3. 编写 Markdown 内容
4. 更新嵌入的 JSON 数据
5. 保存到对应的目录

## 📊 字段说明

每个模板都包含了详细的字段说明和示例，请参考具体模板文件。
